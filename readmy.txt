1. Установи виртуальное окружение:

Открой терминал или командную строку.
Перейди в папку с твоим проектом.
Создай виртуальное окружение командой:

python -m venv venv

Активируй виртуальное окружение:
На Windows:

venv\Scripts\activate

Установи зависимости:


Убедись, что файл requirements.txt находится в корне твоего проекта. 
Этот файл содержит список всех необходимых пакетов.
Установи зависимости командой:

pip install -r requirements.txt

Применить миграции:

Django использует миграции для настройки базы данных. Выполни команду:

python manage.py migrate


Создай суперпользователя (необязательно, но полезно для администрирования):

Выполни команду:

python manage.py createsuperuser


Следуй инструкциям на экране для создания учетной записи суперпользователя.
Запусти сервер разработки:

Выполни команду:

python manage.py runserver


Если у тебя нет файла requirements.txt, ты можешь создать его самостоятельно. 
Для этого нужно выполнить несколько шагов:

Создай и активируй виртуальное окружение (если еще не сделано):

python -m venv venv
venv\Scripts\activate      # для Windows
source venv/bin/activate   # для MacOS и Linux


Установи необходимые пакеты:

Установи Django и другие пакеты, 
которые могут понадобиться для твоего проекта. Например, если ты используешь только Django:

pip install django

Если тебе нужны другие пакеты, установи их тоже. Например:

pip install djangorestframework
pip install pillow

Создай файл requirements.txt:

После установки всех нужных пакетов, сгенерируй файл requirements.txt:

pip freeze > requirements.txt

Проверь файл requirements.txt:

Убедись, что файл requirements.txt содержит все установленные пакеты. Пример содержимого файла:

Django==3.2.5
djangorestframework==3.12.4
Pillow==8.2.0


Применение миграций:

python manage.py migrate

Создание суперпользователя:

python manage.py createsuperuser


Запуск сервера:

python manage.py runserver
